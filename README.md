# Termux_Style


## How to install

Follow the steps below - 

```bash
# go to home dir - 
cd $HOME

# clone this repository - 
git clone https://github.com/Rocket-Bhaiya/Termux_Style.git

# change to Termux_Style dir -
cd Termux_Style

```

## Run

